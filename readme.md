## Projects

### 1. Student Grades Analysis

Analysis of student performance using NumPy arrays.

#### Features

- Compute averages
- Find minimum and maximum scores
- Analyze performance by student
- Analyze performance by subject
- Apply bonus points
- Create Boolean masks

#### Concepts Practiced

- ndarray
- shape
- axis
- mean()
- min()
- max()
- Broadcasting
- np.where()

---

### 2. Mini Image Processing Engine

Image manipulation using NumPy arrays.

#### Features

- Brightness adjustment
- Image normalization
- Horizontal flipping
- Transposition
- Thresholding
- RGB manipulation

#### Concepts Practiced

- Matrix operations
- Image representation
- Slicing
- Broadcasting
- Transpose
- Flip
- Boolean masks

### 3. Vectorized Stock Market Simulation

Simulation of stock price movements using NumPy.

#### Features

- Generate daily returns
- Simulate hundreds of stocks
- Compute cumulative performance
- Analyze volatility
- Rank top-performing assets
- Profitability analysis

#### Concepts Practiced

- Vectorization
- Broadcasting
- Aggregation methods
- Random generation
- cumsum()
- cumprod()
- Statistical analysis
